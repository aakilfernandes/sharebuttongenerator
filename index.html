Share Button Generator

I got tired of looking up sharing APIs everytime I wanted some simple share buttons, so I built this tool. You can grab the whole project off of Github.

Installation

  1. Include Font Awesome if you haven't already <link rel="stylesheet" href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.min.css"/>
  2. Download and include the css and js
    • CSS: sbg.css
    • JS: sbg.js
  3. Include the HTML from the generator
  4. Call sbg() after the HTML (and necessary AJAX calls)
